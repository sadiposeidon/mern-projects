# MERN Layihəsi

Bu layihə **MERN stack** (MongoDB, Express.js, React, Node.js) istifadə edilərək hazırlanıb.

## Tələblər

- Node.js və npm
- MongoDB
- Tailwind

## Quraşdırma

1. Depoyu klonlayın:
    ```bash
    git clone <repo-link>
    cd mern-projects
    ```

2. Backend üçün:
    ```bash
    cd backend
    npm install
    npm start
    ```

3. Frontend üçün:
    ```bash
    cd frontend
    npm install
    npm start
    ```

## İstifadə

- Backend serveri `http://localhost:5000` ünvanında işləyir.
- Frontend isə `http://localhost:3000` ünvanında əlçatandır.
